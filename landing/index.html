Install

Run Locally in Minutes

No cloud accounts, no API keys, no data leaks. Pick your method.

# Install globally $ npm install -g trakshya-waf # Launch the dashboard $ trakshya-waf

Requires Node.js 18+. The package downloads the latest AppImage on install. ~75MB download.

# One-liner install $ curl -fsSL https://trakshya.waf/install.sh | bash # Launch $ trakshya-waf

The install script detects your OS/arch and downloads the correct AppImage from GitHub Releases. ~75MB download.

# Download from GitHub Releases $ curl -L https://github.com/Pradyu12/KALKI-WAF/releases/latest/download/Trakshya-WAF-x86_64.AppImage -o trakshya-waf.AppImage $ chmod +x trakshya-waf.AppImage # Run $ ./trakshya-waf.AppImage

Distributed as a single AppImage. No dependencies required. Works on Ubuntu 22.04+, Fedora 38+, Debian 12+.

# Clone the repo $ git clone https://github.com/Pradyu12/KALKI-WAF.git $ cd KALKI-WAF # Build and run all services $ docker compose up --build # Dashboard at http://localhost:8000

Starts the Rust proxy, Go API, C monitor, and dashboard. Great for development and CI environments.
